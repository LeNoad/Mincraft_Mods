# Mincraft_Mods

# 명령어

1. #back
죽은 장소로 돌아갑니다. ( 서버에 처음 접속시 UUID 등록 )

2. #set [저장할 이름]
저장하고 싶은 장소를 저장한다.

3. #home [저장한 이름]
저장했던 장소로 텔레포트 한다.

4. #list
지금까지 저장하였던 장소들을 보여준다.
